# autoRigger

### How to Use

### 0. Unzip the file
### 1. Put the folder in Maya Script Directory. Usually 'C:\Users\user\Documents\maya\scripts' on Windows, and 'Library/preferences/autodesk/maya/scripts' for Mac OS.
### 2. Open Maya Script Editor
### 3. Execute Following Code

import Kaia_AutoRigger
from importlib import reload
reload (Kaia_AutoRigger)

Kaia_AutoRigger.AutoRigFace()
